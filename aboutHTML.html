ABOUT

Surreality is my first game project made in the Unity engine. The game started development in July 2014 after many revisions of ideas, and I hope to use it in a portfolio for game development. Unity is a great engine, so if you want to make a game give it a try:

I like games with weird and cool looking mechanics, which I have tried to do with this game. Some of my favourites are Portal, Katamari Damacy and Antichamber.
